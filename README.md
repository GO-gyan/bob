# BOB

## Installation

* clone project
```
git clone https://github.com/GO345724/bob.git
```
* install webpack globally
```
npm install -g webpack
```
* install package.json dependencies
```
npm install
```
* install bower globally
```
npm install -g bower
```
* install bower dependencies
```
bower install
```
* run project
```
npm run serve
```
